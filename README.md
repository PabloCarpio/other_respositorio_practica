# LabHub - Repositorio Web de Laboratorios Académicos

## 📋 Definición del Sistema (Tarea 1)

**Nombre del sistema:** LabHub  
**Objetivo breve:** Desarrollar una aplicación web que simule un repositorio digital tipo GitHub, permitiendo el registro, visualización, búsqueda, filtrado y descarga de laboratorios académicos de forma sencilla y profesional.  
**Usuarios del sistema:** Estudiantes y profesores universitarios (principalmente de carreras de Ingeniería, Ciencias y Tecnología).

---

## ✨ Características Principales

- Diseño moderno estilo **GitHub** (dark mode)
- Inicio de sesión visual
- Subida de laboratorios con **drag & drop** real de archivos
- Listado de repositorios en tarjetas interactivas
- Buscador en tiempo real por nombre
- Filtros por categoría
- Descarga simulada de archivos
- Persistencia de datos con LocalStorage
- Totalmente responsive (celular, tablet y PC)
- Código limpio y separado (HTML + CSS + JS)

---

## 🛠 Tecnologías Utilizadas

| Tecnología       | Uso                          |
|------------------|------------------------------|
| HTML5            | Estructura de las páginas    |
| Tailwind CSS     | Estilos y diseño responsivo  |
| JavaScript (ES6) | Lógica completa del sistema  |
| LocalStorage     | Persistencia de datos        |

## 📁 Estructura del Proyecto
LabHub/
├── index.html          ← Página principal (listado de repositorios)

├── css/

├── js/
│   ├── data.js         ← Datos iniciales (10 laboratorios)
│   └── app.js          ← Toda la lógica JavaScript
└── README.md


---

## 🚀 Cómo Ejecutar el Proyecto

1. Descarga o clona la carpeta completa.
2. Abre la carpeta en tu explorador de archivos.
3. Haz doble clic en **`login.html`** (o ábrelo con Live Server).
4. Inicia sesión con cualquier usuario (ejemplo: `pablo.estudiante`).
5. ¡Listo! Ya puedes explorar, buscar, filtrar y subir nuevos laboratorios.
